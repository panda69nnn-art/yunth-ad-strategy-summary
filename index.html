FOR AD PRODUCTION

Yunthは、
誰にどう届けるべきか。

公式LP、ぎいちゃんのプロデュース資料、note記事「なぜYunthは選ばれているのか」を統合した、動画広告制作のための共有メモ。 きれいな美容液広告ではなく、迷っていた人に「今なら試す理由」を作る広告として整理する。

30秒 使用期限という強い記憶フック。個包装の必然性を作る。
1,980円 気になっていた人の背中を押す初回価格。特典とセットで効く。
30-44歳 濃い悩みより「このままだと不安」が強い予防美容層。

01 Target

一番届ける相手

「シミに深く悩んでいる人」ではなく、「夏前に早めのケアをしたいけど、Yunthを買う理由がまだ足りなかった人」。 悩みの強さより、購入直前の迷いをほどく方が勝ちやすい。

主対象は、Yunthが気になっていたけど、高いし定期も不安で止まっていた30-44歳。

夏前のシミ予防 生ビタミンCが気になる 失敗したくない 価格で迷っていた 忙しい朝でも続けたい
39 理央さんタイプ / 会社員・家庭あり・美容液は慎重に選ぶ

購買スイッチ

  • 日差しを浴びた日の肌不安、メイク前のくすみ印象、ハリツヤ不足が気になり始めている。
  • Yunthは見たことがあり、商品名もなんとなく知っている。けれど通常価格や定期購入で止まっていた。
  • 「使用期限30秒の生VCを、毎回1包ずつ開けたてで使える」と聞くと、価格に納得感が出る。
  • 最後に初回1,980円、送料無料、特典つきで「今なら試していいかも」になる。

Primary

30-44歳。夏前にシミ対策を始めたいが、美白美容液選びに迷っている人。

Secondary

個包装・衛生感・持ち運び・使い切りに価値を感じる人。旅行やジム導線も合う。

Not Main

濃いシミを今すぐ消したい人、刺激ゼロを最優先する人。期待値がズレやすい。

02 Why Chosen

note記事から吸うべきこと

記事のポイントは、Yunthが「バズったから売れた」ではなく、商品・言葉・SNS接点・デザインが同じ方向を向いているブランドだということ。 広告では、この構造を壊さずに購入理由へ接続する。

商品力が説明を支える

個包装、使用期限30秒、水不使用、アスコルビン酸。珍しさではなく、フレッシュに使う理由として見せる。

SNSで判断材料が揃う

購入前にテクスチャー、使用感、合う人・合わない人を想像できる。広告は初見説明より「見たことある」を購入へ進める役。

言葉が迷わせない

「生VC」「使用期限30秒」「水不使用」のように、短くて意味が伝わる言葉を使う。難しい成分説明に逃げない。

引き算のデザイン

余白、視線誘導、同じ商品画像の反復で世界観を作る。広告も情報を盛るほど、Yunthらしさから遠ざかる。

広告に翻訳すると

Yunthは「知らない商品をゼロから説明する広告」より、すでに少し知っている人へ “なぜ今買っていいのか”を渡す広告が強い。美しい世界観だけで終わらせず、最後は価格・特典・期限で行動理由を作る。

03 Winning Route

勝ち導線

LPとぎいちゃん資料の芯は、悩みを選ばせてから商品理解へ入り、最後にオファーで背中を押すこと。 きれいな肌映像だけではなく、パウチ・価格・特典を買う理由として見せる。

0-3秒
違和感フックで止める 「Yunth、高いと思ってた人へ」「夏前にシミを作らせないケア」など、今の迷いに刺す。
3-12秒
悩みを一つ選ばせる シミ予防、くすみ印象、乾燥による毛穴目立ち。濃い悩みではなく予防意識へ寄せる。
12-28秒
生VCと個包装の必然性 パウチ開封、液が出る瞬間、肌にのせる直前。個包装は便利さではなく鮮度の証拠。
28-45秒
医薬部外品の価値を正確に メラニンの生成を抑え、しみ・そばかすを防ぐ。既存のシミが消える話にはしない。
45-65秒
本品と特典を一気に見せる 箱、パウチ、シートマスク、化粧水パウチ、レチノールなど。物量で「今買う理由」を作る。
65-90秒
価格・期限・CTA 初回1,980円、50%OFF、送料無料。定期条件は簡単と言い切らず、詳細はLP確認へ。

04 Creative Design

動画広告の作り方

生VC鮮度型を本命にして、季節・価格・復帰購入・悩み別へ横展開する。画のピントはモデルの美肌より、パウチとテクスチャー。

ルート 冒頭 刺す不安 撮るべき画
本命
生VC鮮度型		 生ビタミンCを、毎回開けたてで。 高い美容液なら、ちゃんとフレッシュな状態で使いたい。 1包を持つ指、切り口、液が出る瞬間、肌にのせる直前。
強い
価格迷い型		 Yunth、高いと思ってた人へ。 気になっていたけど、通常価格や定期で止まっていた。 価格、特典、本品を並べる。押し売りより「今なら試しやすい」。
季節
夏前シミ予防型		 夏に向けて、シミを作らせないケア。 日差しが強くなる前に、何か始めた方がいい気がする。 朝の洗面台、窓辺の日差し、頬を見る自然な動き。
横展開
悩み別LP直結型		 くすみ印象、乾燥毛穴、シミ対策で字幕差し替え。 肌悩みは一つではない。自分に関係ある入口が欲しい。 鼻横、頬、口元、メイク前の肌確認。超接写比較は避ける。

冒頭コピー案

Yunth、高いと思ってた人へ。

使用期限30秒。だから、1包ずつ。

夏前のシミ対策、洗顔後に+1。

個包装は便利さじゃなく、鮮度のため。

画づくりのルール

  • パウチを切る音、液が出る音、手のひらの質感を撮る。
  • 肌は白く加工しすぎず、自然光で「朝のケア」に寄せる。
  • 注釈スペースを必ず空ける。美白、浸透、毛穴、ツヤは文脈が命。
  • 価格と特典は最後に整理。途中で売り込みすぎると商品理解が浅くなる。

05 Expression

使う言葉・避ける言葉

Yunthは言葉が強い商品なので、盛るより削る。薬機法・景表法上も、期待値を上げすぎず、公式表現に寄せる。

使いやすい

  • 生ビタミンC / 有効成分アスコルビン酸
  • 使用期限30秒という発想
  • 毎回開けたて / 1回分ずつ
  • 水を1滴も含まない濃密な一滴
  • メラニンの生成を抑え、しみ・そばかすを防ぐ
  • 洗顔後、化粧水の前に+1
  • 初回1,980円 / 50%OFF / 送料無料 / 特典つき

避ける

  • シミが消える / 薄くなる
  • 即効で白くなる
  • 毛穴がなくなる
  • 酸化しない / 劣化しない
  • 刺激ゼロ / 敏感肌でも絶対安心
  • 無臭 / 軽い美容液 / さらさら
  • 30秒を過ぎると使えない
  • 定期でも簡単にやめられると言い切る
